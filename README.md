## Overview
BetterHitreg is an **open source** client-side Fabric mod for Minecraft **1.19.4 – 1.21.11** that registers hits **before the server** making it feel as if you were low ping

---

## Extras
- Various utility features mainly designed for sword pvp
- Includes a keybind to **switch main hand** & keybinds to track **pvp scores like in tier tests**
- UI & Render colors can be edited via Hitreg.properties in your instances config folder

---

## Safety
- ✅ **Allowed on mctiers.com & pvptiers.com for tier testing** (version 1.0.4 confirmed, if you believe this is incorrect please make a github issue)
- ✅ **No known false flags or punishments from any anticheat** have been recorded and no features are designed to provide any unfair technical advantage
- ✅ All custom hits are in **appearance only**, your actual hits are **not modified** in any way, only the way they are rendered and sound is affected

---

## Performance
- Starting with version 1.0.5, almost no processing occurs on the network thread. Even with earlier versions the mod shouldn't impact your network performance in any noticeable way.
- Rendering (like hitboxes and crosshairs) will cause a substantial performance drop on versions older than 1.21.11, If you care about performance and use these features, it's recommended to use version 1.21.11
